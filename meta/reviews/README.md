# Reviews

Structured code-review documents produced by the `pouk-ai-reviewer` agent for every change shipped by `pouk-ai-engineer`. Each review assesses one diff against the masterplan, the governing PM spec(s), and engineering quality bars, and ends with a recommendation that Arian uses to decide on merge.

Filename convention: `YYYY-MM-DD-<short-slug>.md` (slug = branch name, PR title, or feature being reviewed).
